Esto te permitirá volcar tus discos a un formato que puedas instalar con el fin de jugar sin necesidad de tener el disco.
{: .notice--info}

No puedes utilizar disc2app en RedNAND para volcar el juego, pero si puedes instalar el juego volcado en RedNAND. Las instrucciones reflejan esto.
{: .notice--info}

{% capture notice-2 %}
**If you are planning to use an external hard drive to install your dumped discs on, note the following:**

    + You cannot use the same hard drive for vWii and Wii U games.
    + Formatting an external hard drive for use with a Wii U will erase its contents.
    + Si estás usando un disco duro externo que no tiene fuente de alimentación propia, necesitarás un [Y-cable](http://amzn.to/2mjQjin).

{% endcapture %}

<div class="notice--info">{{ notice-2 | markdownify }}</div>

Si planeas utilizar un disco duro externo para instalar tus discos volcados, ten en cuenta que la Wii U sólo puede utilizar hasta 2tb de un disco duro externo, incluso si es de mayor capacidad.
{: .notice--info}

#### Qué necesitas

* Una tarjeta SD que ya contenga una RedNAND y todos los archivos de la página [Comencemos](get-started) en ella
* Se requieren 25gb disponibles en tu tarjeta SD

#### Instrucciones

##### Sección I - Formatear el disco externo

Si planeas utilizar un disco duro externo para instalar tus discos volcados, sigue esta sección.
{: .notice--info}

Si no planeas utilizar un disco duro externo o ya tienes uno formateado para Wii U, salta esta sección.
{: .notice--info}

¡Ten en cuenta que esto borrará completamente el contenido del disco duro externo!
{: .notice--danger}

1. Conecta tu HDD USB en tu Wii U mientras está apagada
2. Enciende tu Wii U
3. Formatea el disco duro externo cuando se te pida

##### Sección II - Iniciar disc2app en SysNAND

1. Extrae cualquier disco que pueda estar en la unidad de disco de la Wii U
2. Inicia disc2app en la SysNAND usando el método correspondiente a tu instalación de Mocha CFW

###### CBHC

1. Apaga tu Wii U
2. Hold (Home) on boot to enter the CBHC boot menu
3. Selecciona "Boot Homebrew Launcher" para iniciar el Homebrew Launcher en SysNAND
4. Inicia disc2app

###### Haxchi

1. Mantén (X) mientras inicias tu juego de la consola virtual de DS para iniciar disc2app

###### Homebrew Launcher

1. Enter the Homebrew Launcher on SysNAND using `http://loadiine.ovh`
2. Launch disc2app

##### Sección II - Usando disc2app

1. Press (A) to dump to the SD card
2. Insert the disc you want to dump when prompted
3. Allow the disc dump process to proceed automatically; this will take some time
3. When it is complete, you will be returned to the SysNAND home menu
  + CBHC users may be returned to RedNAND instead

##### Sección III - Instalar el juego volcado

1. Boot RedNAND using the method of your choice
2. Launch the Homebrew Channel on RedNAND to enter the Homebrew Launcher
3. Inicia WUP Installer - Mod Y
4. Install the game to your desired destination
  + Press (A) to install to the system memory
  + Press (X) to install to the external hard drive
5. Allow the installation process to proceed automatically; this will take some time
6. When it is complete, you will be returned to the Homebrew Launcher
7. Press (Home) to return to the RedNAND home menu

___

To reclaim the space on your SD card taken up by the disc dump, delete the newly created folder in `/install/` on your SD card
{: .notice--info}

If everything went well, you'll see your new game installed on the menu ready to be played!
{: .notice--success}

You can ensure that the game installed to the correct directory by checking its installation location in the System Settings under "Data Management"
{: .notice--success}
